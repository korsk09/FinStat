# FinStat
Группа: 9-4-ПО-22
Студенты: Аминов Руслан Радикович, Давыдок Дарья Игоревна

Краткое описание:
Приложение FinStat - это лёгкий и удобный помощник для любителей следить за курсом валют. В нём можно легко и просто конверитировать нужные пользователю валюты, отслеживать как они изменялись в цене и т.д. Оно решает такие проблемы как: разная информация на сайтах и экономию времени. Это приложение создано для тех, кто часто пользуется информацияй о ценнах валют, финансовых аналитиков, предпренимателей и так далее.
